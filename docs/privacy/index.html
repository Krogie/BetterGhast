Privacy Policy

Last updated: April 1, 2026

1. What Data We Collect

BetterGhast only collects data that is strictly necessary for its functionality:

Data Purpose Stored
Guild (Server) ID Associate tags with the correct server Yes, in database
Tag content & keywords Store and display tags when triggered Yes, in database
Usage counters Anonymous tag usage statistics Yes, in database

2. What We Do NOT Collect

  • We do not collect or store user messages outside of tag commands
  • We do not collect usernames, user IDs, or any personal information
  • We do not collect message history or chat logs
  • We do not track individual user behavior or activity
  • We do not sell, share, or transfer any data to third parties

3. How Data Is Stored

All data is stored in a private MariaDB database on a secured server. Access is restricted to the bot operator. Data is transmitted over encrypted connections.

4. Data Retention

Tag data is stored as long as the bot is active on your server. When the bot is removed from a server, you may request deletion of all associated data.

5. Data Deletion

You can delete individual tags at any time using the /tags manage command with the remove option. To request complete deletion of all data associated with your server, contact us via GitHub Issues.

6. Third-Party Services

BetterGhast interacts with the following third-party service:

  • Discord API — Required for bot operation. Subject to Discord's Privacy Policy.

7. Children's Privacy

BetterGhast does not knowingly collect data from children under 13. Use of Discord (and therefore BetterGhast) requires compliance with Discord's minimum age requirements.

8. Changes to This Policy

We may update this privacy policy at any time. Changes will be reflected by the "Last updated" date above.

9. Contact

For privacy-related questions or data deletion requests, reach out via GitHub Issues.
